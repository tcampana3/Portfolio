Projects

E-commerce Store Development (Shop Fusion)

Developed a single-page e-commerce application using React.js, Node.js, Express.js, and MongoDB. The project includes a responsive user interface designed with Bootstrap and CSS. Key features include product display pages, a shopping cart, a checkout process, and an Admin panel for CRUD operations on products. Cyclone Carpool - Mobile Application

Developed a rideshare mobile application, with a team of four software engineering students, for university students using Java, Spring Boot, Android Studio, and MySQL. Designed the user interface with Figma and implemented key features such as trip management, ID verification, location tracking, payment requesting/processing and real-time messaging with WebSockets. Built and tested RESTful APIs to enable seamless communication between frontend and backend components, ensuring secure and reliable data handling.

Multi-Store Retail Operations System

Engineered a multi-store retail operations system using Java, with CSV file storage for lightweight data persistence. Implemented key functionalities including sales processing, returns, inventory tracking, rentals, warranties, and customer feedback management. Leveraged Facade and Observer design patterns to enhance scalability, simplify workflows, and automate event-driven tasks. Archived Message Reconstruction

Built a Java-based binary-tree decoder to reconstruct archived messages from compressed files. Utilized bitwise operations and recursive tree construction to decode messages efficiently. This project also included calculating message-specific statistics, such as average bits per character and space savings, as bonus features. This project enhanced skills in Java and binary-tree data structures.

LEGv8 Disassembler for Machine-Level Programming

Created a Java-based disassembler to convert binary LEGv8 instructions into assembly language. Managed endianness issues and optimized the process using bitwise operations extensively. Additionally, developed shell scripts to automate the build and execution processes, improving reliability and ease of use for the disassembler. This project involved Java programming and low-level machine language concepts.

2D Point Sorting and Median Calculation

Developed a Java program to sort 2D integer points using selection sort, insertion sort, merge sort, and quicksort. Calculated the median point based on sorted coordinates and compared the execution times of each sorting algorithm to analyze efficiency. This project strengthened understanding of Java programming, sorting algorithms, and computational efficiency.

Profit Utilization for Internet Service

Simulated customer behavior on an n x m grid, developing Java classes to calculate yearly profit utilization for internet service. This project involved modeling and data analysis in Java to understand user behavior and optimize service distribution.

Dijkstra’s Algorithm Implementation

Implemented Dijkstra’s algorithm in Java to compute the shortest path from a single source to all vertices in a directed graph without negative weights. Leveraged priority queues for efficient processing, optimizing the algorithm to handle real-world applications involving complex networks. This project deepened understanding of graph theory and algorithmic optimization.

Stout List - Custom Doubly Linked List Implementation

Developed a specialized doubly linked list, called the Stout List, to handle variable node capacities. Each node in this list can hold multiple elements, allowing for an efficient structure that minimizes memory overhead by reducing the number of node pointers. Implemented complex features including custom iterators for traversing and modifying the list and an insertion sort algorithm for sorting list elements. This project was built in Java and involved in-depth work with AbstractSequentialList and ListIterator interfaces.
